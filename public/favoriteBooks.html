What is this website?

This webpage is created for the "Web Development" course at Athens University of Economics and Business. It implements a web application using the Fetch API, which gives the user the chance to search for books from the Penguin Random House electronic "bookstore" API. You can learn more about Penguin Random House following the links on the above horizontal menu. The search results are printed by dynamically producing HTML code using the DOM API properties or the Handlebars library and appropriate HTML templates.
